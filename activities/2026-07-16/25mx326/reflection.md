# Reflection



As a software engineer, my daily work usually looks like this:

1. Attend Daily Stand-up.
2. Pick a Jira task.
3. Create a Git branch.
4. Develop the feature.
5. Commit and push code.
6. Raise a Pull Request.
7. Address review comments.
8. QA tests the feature.
9. Fix bugs if needed.
10. Merge and deploy.
11. Repeat in the next sprint.
